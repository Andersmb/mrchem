# Change log

## Version 1.0.0-alpha1 2020-05-05

### Features

- Hartree-Fock
- Kohn-Sham DFT (LDA/GGA/hybrid)
- Restricted (closed-shell) and unrestricted
- External electric field
- Ground state energy
- Dipole moment
- Quadrupole moment
- Polarizability (static/dynamic)
- Magnetizability
- NMR shielding
- Density/orbital plots
